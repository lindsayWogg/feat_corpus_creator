-----------------GUIDE D'INSTALLATION--------------

AFIN DE D'INSTALLER LES DÉPENDANCES NÉCÉSSAIRE AU LANCEMENT DE CETTE APPLICATION
VOUS DEVEZ INSTALLER LES BIBLIOTHÈQUES INSCRITS DANS LE FICHIER "requirements.txt"
DEUX (2) METHODES S'OFFRENT À VOUS:
METHODE 1: LANCER L'INSTALLATION DE CHAQUE BIBLIOTHÈQUE MANUELLEMENT DANS LE TERMINAL
EXEMPLE:  pip install ma_bibliotheque

METHODE 2: LANCER LA COMMANDE SUIVANTE DANS LE TERMINAL
EXEMPLE pip install -r"chemin_du_requirements.txt"

--------------------BASE DE DONNÉES
IL VOUS SUFFIT D'EXECUTER LE SCRIPT SQL DANS LE FICHIER "taln.sql"
---------------------------lancement de l'application
POUR LANCER L'APPLICATION VOUS DEVEZ LANCER LE FICHIER "main.py"
DEUX (2) METHODES S'OFFRENT À VOUS:
METHODE 1: LANCER LE SCRIPT VIA VOTRE EDITEUR DE CODE
METHODE 2: LANCER LA COMMANDE SUIVANTE DANS LE TERMINAL
-OUVREZ UN TERMINAL DANS LE DOSSIER CONTENANT LE FICHIER "main.py"
-LANCER LA COMMANDE SUIVANTE:
==>python main.py 
ou ==>python3 main.py
